# CodeAlpha_StockTradingPlatform

## Description

Stock Trading Platform is a Java console-based application developed for the CodeAlpha Java Programming Internship. The system allows users to view stock market data, buy and sell stocks, track portfolio performance, and store transaction records using File I/O.

## Features

* View Market Data
* Buy Stocks
* Sell Stocks
* Portfolio Tracking
* Portfolio Value Calculation
* Transaction Storage using File I/O
* Console-Based Interface

## Technologies Used

* Java
* File I/O
* Object-Oriented Programming (OOP)

## How to Run

Compile:

javac StockTradingPlatform.java

Run:

java StockTradingPlatform
